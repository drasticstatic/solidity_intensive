🧪 Solidity Intensive

A complete language reference — 18 contract modules covering every core Solidity concept from primitives to DeFi interactions.

Solidity ^0.8 Hardhat Tested 18 Modules DAPPU Bootcamp

📖 About This Reference

Each contract in this repo isolates a single Solidity concept with clean, commented code and corresponding Hardhat tests. Built during an intensive blockchain developer bootcamp — think of it as a Solidity cookbook where every chapter is a standalone, copy-ready recipe.
